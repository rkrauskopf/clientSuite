Browser/Mobile Client details

A platform for developing rich hybrid mobile apps using Cordova, Angular.js and Bootstrap.

Prerequisites

To build and run the code, you will need to do a few simple installs to get started.

  1. Node.js is required to install the necessary libraries to host the client.
    • Node ships with NPM (the Node Package Manager), but not necessarily the latest version. So after installing Node, you should upgrade to the latest version of NPM using the following command: 
      npm install -g npm
    • Windows systems will continue running the old version of NPM, until you manually delete the old version. So you need to navigate in the windows command line to the following directory:
      • C:\Program Files\nodejs
      and run the command 
      npm install npm

  2. Bower is require to install highcharts.js and jQuery libraries for now
